Projects

  • Gray-box Data Leakage Detection in LLM Benchmarks

    Investigated the reliability of benchmark evaluation under opaque pre-training data conditions.

    • Generated derivative benchmark sequences for MMLU and CMMLU through multiple-choice option restructuring.
    • Queried target models such as LLaMA 2 and Qwen for log-probability distributions.
    • Applied Isolation Forest to identify contaminated cases under deliberate option scrambling.
  • SR-TOD for Micro-Object Detection in UAV Imagery

    Built a high-precision visual detection pipeline for tiny objects under waves, glare, and uneven illumination.

    • Combined SuperGlue-based stitching with weighted fusion and Multi-Scale Retinex preprocessing.
    • Proposed the Self-Reconstruction Tiny Object Detection framework with a lightweight reconstruction head.
    • Designed the Difference-Guided Feature Enhancement module to improve multi-scale feature discrimination.
  • Autonomous Driving U-turn Decision and Path Optimization

    Constructed a perception-prediction-decision-planning-control pipeline for complex urban U-turn scenarios.

    • Predicted dynamic obstacles with Kalman Filter and quantified safety margins with Time-to-Collision.
    • Planned smooth trajectories with cubic Bezier curves and Analytic Hierarchy Process.
    • Used Model Predictive Control for low-risk trajectory refinement and high-risk emergency braking.
