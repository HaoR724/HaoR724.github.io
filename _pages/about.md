      <p class="home-section-label">Selected Research</p>
      <h2 class="home-section-title">Three current directions that define the homepage.</h2>
    </div>
    <div class="home-project-grid">
      <a class="home-project" href="{% link _projects/1_llm_leakage.md %}">
        <p class="home-project__meta">Independent Research / Core Member</p>
        <h3>Gray-box Data Leakage Detection in LLM Benchmarks</h3>
        <p>
          Investigated benchmark contamination under opaque pre-training data by restructuring multiple-choice options
          and analyzing log-probability distributions.
        </p>
        <p>
          The method uses Isolation Forest to identify contamination in difficult cases where answer options are
          deliberately scrambled.
        </p>
      </a>
      <a class="home-project" href="{% link _projects/3_uav_vision.md %}">
        <p class="home-project__meta">University Innovation Project / Core Developer</p>
        <h3>SR-TOD for Micro-Object Detection in UAV Imagery</h3>
        <p>
          Designed a high-precision visual detection framework for tiny targets under waves, glare, and complex
          background clutter in maritime aerial imagery.
        </p>
        <p>
          The pipeline combines SuperGlue-based stitching, Multi-Scale Retinex preprocessing, SR-TOD, and the DGFE
          module for sharper multi-scale feature representation.
        </p>
      </a>
      <a class="home-project" href="{% link _projects/2_autonomous_driving.md %}">
        <p class="home-project__meta">Core Algorithm Designer</p>
        <h3>Autonomous Driving U-turn Decision and Path Optimization</h3>
        <p>
          Built an integrated perception-prediction-decision-planning-control framework for complex two-way urban
          U-turn scenarios.
        </p>
        <p>
          The system combines Kalman Filter prediction, Time-to-Collision safety modeling, Bezier trajectory planning,
          and Model Predictive Control for risk-aware execution.
        </p>
      </a>
    </div>
  </section>
