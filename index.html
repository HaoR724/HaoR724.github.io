Selected Research

Three current directions that define the homepage.

Independent Research / Core Member

Gray-box Data Leakage Detection in LLM Benchmarks

Investigated benchmark contamination under opaque pre-training data by restructuring multiple-choice options and analyzing log-probability distributions.

The method uses Isolation Forest to identify contamination in difficult cases where answer options are deliberately scrambled.

University Innovation Project / Core Developer

SR-TOD for Micro-Object Detection in UAV Imagery

Designed a high-precision visual detection framework for tiny targets under waves, glare, and complex background clutter in maritime aerial imagery.

The pipeline combines SuperGlue-based stitching, Multi-Scale Retinex preprocessing, SR-TOD, and the DGFE module for sharper multi-scale feature representation.

Core Algorithm Designer

Autonomous Driving U-turn Decision and Path Optimization

Built an integrated perception-prediction-decision-planning-control framework for complex two-way urban U-turn scenarios.

The system combines Kalman Filter prediction, Time-to-Collision safety modeling, Bezier trajectory planning, and Model Predictive Control for risk-aware execution.
